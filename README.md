# gnome-oled-mask
GNOME extension to minimize OLED burn-in by applying a rectangular pixel-shift mask.

## Tested on
- Gnome 49 (Linux 6.19.8-1-cachyos)

## Setup
- Move "oled-mask@gludwig" folder into "~/.local/share/gnome-shell/extensions" folder.
- Enable and change settings with the "Extensions" or "Extensions Manager" application.
